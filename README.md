[ระบบแนะนำเพลงตามความรู้สึก♥️]
 แอปพลิเคชันนี้พัฒนาด้วย Streamlit เพื่อวิเคราะห์อารมณ์ของผู้ใช้จากข้อความที่พิมพ์ และแนะนำเพลงที่เหมาะสมกับอารมณ์นั้นโดยอัตโนมัติ ภายในหน้าเดียวใช้งานง่าย กลุ่มเป้าหมายคือผู้ที่ต้องการฟังเพลงให้เข้ากับความรู้สึกในช่วงเวลานั้น ๆ แอปช่วยลดขั้นตอนการค้นหาเพลงเอง ทำให้ผู้ใช้สามารถค้นพบเพลงที่ตรงกับอารมณ์ได้สะดวก รวดเร็ว และสนุกมากยิ่งขึ้น

[ฟีเจอร์หลัก]
  •	ฟีเจอร์ที่ 1 วิเคราะห์อารมณ์ของผู้ใช้จากข้อความที่พิมพ์ โดยใช้การตรวจจับคำสำคัญ (Keyword Matching)
	•	ฟีเจอร์ที่ 2 จัดหมวดหมู่เพลงตามอารมณ์ เช่น อินเลิฟ เหนื่อย เบื่อ หรืออกหัก
	•	ฟีเจอร์ที่ 3 สุ่มแนะนำเพลงจำนวน 50 เพลงต่อหนึ่งอารมณ์ พร้อมลิงก์เปิดบน YouTube
	•	ฟีเจอร์ที่ 4 แสดงตัวนับลำดับเพลง (เช่น เพลงที่ 1 / 50) และรีเซ็ตการสุ่มใหม่อัตโนมัติเมื่อครบ 50 เพลง
	•	ฟีเจอร์ที่ 5 ใช้ Session State เพื่อจดจำสถานะของอารมณ์และลำดับเพลงระหว่างการกดปุ่ม
	•	ฟีเจอร์ที่ 6 ออกแบบหน้าเว็บด้วยโทนสีพาสเทล พร้อมการ์ดแสดงผลเพลงเพื่อเพิ่มความสวยงามและประสบการณ์การใช้งานที่ดี

[เทคโนโลยีที่ใช้]
  •	Language: Python
	•	Framework: Streamlit
	•	Libraries: random, urllib.parse, pandas, Streamlit

โปรเจกต์นี้พัฒนาด้วยภาษา Python และใช้ Streamlit ในการสร้างเว็บแอปพลิเคชันแบบโต้ตอบ (Interactive Web Application) โดยผู้ใช้สามารถพิมพ์ข้อความเพื่อวิเคราะห์อารมณ์ และระบบจะแนะนำเพลงที่เหมาะสมกับความรู้สึกนั้นโดยอัตโนมัติ

ภายในระบบมีการใช้:
	•	random สำหรับสุ่มลำดับเพลง
	•	urllib.parse สำหรับแปลงชื่อเพลงให้เป็น URL ที่ถูกต้องสำหรับการค้นหาบน YouTube
	•	Session State ของ Streamlit เพื่อจัดการสถานะของอารมณ์ รายการเพลง และลำดับการสุ่ม

การเชื่อมต่อกับ YouTube

เมื่อผู้ใช้กดปุ่มสุ่มเพลง ระบบจะ:
	1.	แสดงชื่อเพลง
	2.	ฝังวิดีโอ (Embed) จาก YouTube ภายในหน้าเว็บ
	3.	สร้าง ลิงก์ตรงไปยังหน้า YouTube สำหรับเพลงนั้น เพื่อให้ผู้ใช้สามารถกดเปิดในแท็บใหม่ได้ทันที
การสร้างลิงก์นี้ทำผ่านการเข้ารหัสชื่อเพลงเป็น URL แล้วเชื่อมไปยังหน้า Search ของ YouTube ทำให้ไม่ต้องใช้ YouTube API โดยตรง แต่ยังสามารถเปิดเพลงได้อย่างสะดวกและรวดเร็ว

[สะท้อนความคิด]
โปรเจกต์นี้เป็นการทำเว็บแอปที่ช่วยแนะนำเพลงตามอารมณ์ของผู้ใช้ โดยให้พิมพ์ความรู้สึกลงไป แล้วระบบจะวิเคราะห์และสุ่มเพลงให้ ซึ่งไอเดียเกิดจากการที่บางครั้งเราอยากฟังเพลงให้ตรงกับอารมณ์ แต่ไม่รู้จะเปิดเพลงอะไรดี

ระหว่างทำโปรเจกต์นี้ ได้เรียนรู้หลายอย่างมาก ทั้งการใช้ Streamlit สร้างเว็บแอป การจัดโครงสร้างโค้ดให้เป็นระบบ และการใช้ Session State เพื่อเก็บสถานะของข้อมูล เช่น อารมณ์ปัจจุบัน หรือจำนวนเพลงที่สุ่มไปแล้ว ทำให้เข้าใจว่าการทำเว็บไม่ใช่แค่เขียนโค้ดให้รันได้ แต่ต้องคิดเรื่องลำดับการทำงานของระบบด้วย

อีกอย่างที่ได้เรียนรู้คือการจัดการข้อมูล เช่น การเก็บเพลงเป็นลิสต์ การสุ่มเพลงด้วย random และการสร้างลิงก์ไปยัง YouTube ด้วยการเข้ารหัส URL ทำให้เห็นภาพมากขึ้นว่าเบื้องหลังเว็บหนึ่งเว็บมีรายละเอียดเล็ก ๆ หลายอย่างที่ต้องใส่ใจ

สิ่งที่คิดว่ายังพัฒนาได้อีกคือระบบวิเคราะห์อารมณ์ เพราะตอนนี้ยังใช้วิธีตรวจจับคำสำคัญ ซึ่งอาจไม่แม่นยำมาก ถ้ามีโอกาสพัฒนาต่อ อยากลองใช้เทคนิค AI หรือ NLP เพื่อให้วิเคราะห์อารมณ์ได้ฉลาดขึ้น

โดยรวมแล้ว โปรเจกต์นี้ทำให้เข้าใจขั้นตอนการพัฒนาเว็บแอปตั้งแต่ต้นจนจบ และได้ฝึกคิดเป็นระบบมากขึ้น ถือว่าเป็นประสบการณ์ที่ดีและได้ความรู้เพิ่มขึ้นจริง ๆ
